

<h2> Скрипты </h2>
<b> 1. Выгрузка постов </b>

Выполните следующую команду, находясь в корне репозитория:
```bash
export PYTHONPATH="${PWD}/src"
```

Затем оттуда же выполните:
```bash
python src/scripts/run_loader.py \
  --channel_id "id_канала" \
  --limit "количество_постов" \
  --offset "сколько_пропустить_постов" (опционально, по умолчанию 0)
```
Посты записываются батчами в месте запуска скрипта в csv файл
